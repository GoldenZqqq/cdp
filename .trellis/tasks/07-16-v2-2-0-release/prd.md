# 准备并发布 v2.2.0

## Goal

在 5 个自动化功能任务完成后发布 v2.2.0。

## Requirements

- 更新版本、schema/迁移说明、ReleaseNotes、CHANGELOG、Scoop、官网和双语 README。
- 运行全量 contract、跨平台、安装、性能、安全与发布门禁。
- 统一 push 后等待 CI，再创建 tag、GitHub Release、Gallery 与渠道核验。

## Acceptance Criteria

- [ ] 所有前置子任务各有提交并通过验收。
- [ ] 升级/降级和旧配置兼容证据完整。
- [ ] SHA、tag、Release、Gallery、Scoop、CI 最终核验成功。
