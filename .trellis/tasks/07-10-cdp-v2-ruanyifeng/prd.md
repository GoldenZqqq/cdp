# cdp v2.0 - 项目工作台功能升级

## Goal

将 cdp 从"项目目录切换器"升级为"终端里的项目工作台"，增加项目状态总览、跨平台支持、智能补全等功能，让 cdp 成为开发者管理多项目的核心终端工具。

## Strategy

**产品定位升级**：
- 旧定位："项目目录切换器"
- 新定位："终端里的项目工作台 —— 一个命令看所有仓库状态、切换项目、启动 AI CLI"

**差异化关键点**：zoxide 只知道路径，cdp 知道"项目"——项目的 Git 状态、标签、别名、环境配置、工作区组合。

## Sub-tasks（按优先级排序）

| 优先级 | 任务 | 说明 |
|--------|------|------|
| P0 | cdp status 多项目 Git 仪表盘 | 杀手级差异点，一条命令看所有仓库状态 |
| P0 | macOS 原生支持 | 扩大用户群，覆盖全平台 |
| P1 | Tab 补全 | 成熟 CLI 工具标志，体验质的提升 |
| P1 | 双语 README 重写 | 重新定位为"项目工作台" |
| P2 | Workspace 模式 | 一键启动多项目，强化工作台定位 |
| P2 | 项目环境自动激活 | 类 direnv，进入项目自动执行命令 |
| P3 | 宣传视频重制 | 展示新功能 |
