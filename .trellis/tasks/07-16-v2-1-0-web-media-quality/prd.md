# 增加官网冒烟测试与媒体治理

## Goal

保护官网关键交互和可访问性，并停止媒体资产继续放大 CLI 仓库的克隆成本。

## Requirements

- 浏览器 smoke 覆盖语言切换、tabs、复制、移动导航、Escape、键盘焦点和 reduced motion。
- 检查本地资源引用、JS 语法和基本无障碍语义。
- 建立媒体大小预算与成品/源文件存放策略。
- 不在本任务中重写 Git 历史或擅自删除仍被引用资产。

## Acceptance Criteria

- [ ] 官网 smoke 可在 CI 无头运行。
- [ ] 新媒体超预算时 CI 失败。
- [ ] 重复/过时媒体有迁移方案；任何删除均经过明确范围确认。
