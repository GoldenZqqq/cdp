真正拖慢你的，不是键盘。

是每次开工前，重新寻找上下文。

你记住路径、逐个检查仓库、再重启工具。cdp 把这些零碎动作压成一条可重复的路线。

PATH

长路径不是知识，是负担。

不用记盘符、父目录和项目全名。输入几个字母，唯一匹配时直接到达。

cdp api

STATE

五十个仓库，不该查五十次。

dirty、untracked、ahead、behind 和最近提交时间，一条命令全部展开。

cdp status --dirty

START

工具启动了，上下文却错了。

先进入项目，再启动 Codex、Claude、Gemini、VS Code 或 Cursor。

cdp web -Open claude

不是功能清单，是可执行路线。

一个入口，接管项目上下文。
