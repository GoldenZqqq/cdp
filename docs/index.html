02 / TOOL BOUNDARY

Directory memory and project orchestration solve different jobs.

Use zoxide or autojump for arbitrary visited directories. Use cdp when a named project, repository state, and launcher must travel together.

Tool What it knows Repository overview AI CLI launch
zoxide / autojump Visited directory paths Not built in Not built in
cdp Named projects + local path profiles cdp status cdp api -Open codex

Complementary by design: fast directory recall on one side, repeatable project context on the other.
