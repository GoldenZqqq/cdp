# Asset Descriptions

No website capture was performed. cdp is a local CLI tool.

## Available assets

- **cdp logo**: `docs/assets/cdp-logo-dark-transparent.png` and `cdp-logo-light-transparent.png` in project root
- **Terminal output**: All visuals will be simulated terminal output rendered as styled HTML
- **No screenshots**: CLI tool — all UI is terminal text, best rendered as code blocks with syntax highlighting

## Terminal scenes to simulate

1. Pain point: multiple `cd` commands in rapid succession
2. `cdp status` output table with colored status indicators
3. `cdp api` quick switch with fzf menu
4. `cdp -Open codex` launching AI CLI
5. `cdp workspace fullstack` opening multiple tabs
6. `cdp` + TAB showing completion suggestions
7. Platform icons: Windows / macOS / Linux
8. Install one-liner command
